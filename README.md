<h1>Web Application Security Practice with OWASP ZAP and OWASP Juice Shop</h1>



<h2>Description</h2>
This repository contains my practice work on web application security, focusing on testing and scanning using OWASP ZAP. I used OWASP Juice Shop as the sample web application to explore various security vulnerabilities and testing methodologies.
<br />


<h2>Utilities Used</h2>

- <b>OWASP ZAP</b> 
- <b>OWASP JUICE</b>

<h2>Environments Used </h2>

- <b>Kali Linux</b>
